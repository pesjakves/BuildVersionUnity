# BuildVersionUnity
Utility for Versioning builds in Unity 6.3 . Keeps your builds organized.

CONTENT
1. Toolbar panel that shows the current build number and date. Drop-down allows you to select the bump of build. Button opens build settings.
2. <img width="352" height="125" alt="image" src="https://github.com/user-attachments/assets/fbeb2f19-f5a5-4686-9d8c-10390edb625c" />

3. Adds a new window when clicking Build that allows you to select the type/bump of build.
<img width="520" height="444" alt="image" src="https://github.com/user-attachments/assets/45f3434f-bd49-435d-86a3-878413c4ac91" />

4. Automatic numbers to builds if build was succesful.

FEATURES
- Automatically names your builds with correct lettering, adds a new number based on the type of build.
- The naming convention follows the common format: [major].[minor].[patch].[date].[buildnumber]
- You can manually override the numbering by editing PlayerPrefs
- Makes versioning more predictable and simpler.




**DISCLAIMER**: Works only in the newest 6.3 version of Unity because of the Toolbar Utilities.
If you remove the toolbar editor script, it should work fine in older versions as well.
